getElementsByClassName()

O método getElementsByClassName retorna uma coleção de elementos que possuem uma classe específica. Essa método retorna um HTMLcollection, que é uma coleção semelhante a uma matriz (lista) de elementos HTML. Os elementos de uma coleção podem ser acessados por índice.
