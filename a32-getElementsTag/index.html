getElementsByTagName()

O método getElementsByTagName()retorna uma coleção de todos os elementos que possuem um nome de tag especificado, ou seja, retorna um HTMLcollection (o HTMLcollection é semelhante a uma matriz(lista) de elementos e os elementos de uma coleção podem ser acessados pelo seu índice (0 = valor inicial)).
